# Основні завдання
- Взяти вибірку даних для навчання (рекомендовано — дані з ЛР №3 курсу «Системи штучного інтелекту» або інший відповідний датасет).
- Перейти за інструкцією: https://vc.ru/selectel/41002-instrukciya-sozdanie-neyronnoy-seti-bez-navykov-programmirovaniya
- Виконати всі кроки з налаштування та створення нейронної мережі.
- Побудувати нейронну мережу для розпізнавання об’єктів.
- Провести навчання побудованої моделі.
- Виконати розпізнавання об’єктів за допомогою натренованої нейронної мережі.
